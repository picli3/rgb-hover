# rgb-hover

Controlador para cinta o carga **RGB** basado en MOSFETs de potencia, pensado para conmutación y manejo práctico de iluminación multicolor.

## Propósito del proyecto
Este diseño busca facilitar el control de canales RGB mediante una PCB simple y robusta, adecuada para efectos visuales, tiras LED o sistemas decorativos con cargas relativamente altas.

## Qué incluye
- esquemático del controlador RGB
- PCB en KiCad
- documentación exportada para fabricación y montaje
